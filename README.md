## 概要

1.27mmピッチ2×6Pを2.54mmピッチ1×12Pに変換する基板です。

## 仕様
- ピンピッチ：1.27mm 2×6P ⇔ 2.54mm 1×12P
- 基板サイズ：10mm × 30.48mm
- 基板厚：1.6mm
- 基板色：緑
- 表面仕上げ：無鉛はんだ
- 両面スル―ホール
- 固定穴：25mmピッチ, M3×2穴
